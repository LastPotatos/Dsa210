## Project Overview
Understanding which combat systems in anime receive the most positive reception or are most frequently produced is important, especially now that the genre has gained popularity and become mainstream in the West. This project aims to analyze all fighting anime and compare their ratings based on which combat styles are used. The goal is to have an ML model that can estimate the percentage chance of whether an anime would be a hit or a flop depending on the combat styles it features.
